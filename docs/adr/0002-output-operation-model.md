# ADR 0002: SVG operation model

- Status: accepted
- Date: 2026-07-21

## Context

LightBurn benefits from colour-classified operations, while OpenSCAD 2021.01
does not guarantee portable SVG layers or colour preservation.

## Decision

Models classify cut, engrave, score, and reference geometry. Separate
operation SVG files are authoritative. A composite colour-coded SVG is a
convenience artifact and must be generated by a tested deterministic process.

## Consequences

Fabrication remains reliable without depending on preview colours. Composite
LightBurn imports may require a small export postprocessor.
